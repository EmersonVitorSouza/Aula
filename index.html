Aprenda Git e GitHub para trabalhar em equipe

Passo 1: Criar o Repositório no GitHub

  1. Acesse GitHub e faça login.
  2. Clique no botão + no canto superior direito e escolha New Repository.
  3. Preencha os detalhes:
    • Repository name: Escolha um nome.
    • Adicione uma descrição (opcional).
    • Escolha Public ou Private.
    • Marque Add a README file.
  4. Clique em Create repository.

Passo 2: Conectar ao Repositório no VS Code

  1. Abra o Visual Studio Code.
  2. Pressione Ctrl + Shift + P e digite Git: Clone.
  3. Cole a URL do repositório (encontrada no botão verde Code no GitHub).
  4. Escolha uma pasta no seu computador para salvar o projeto.
  5. Abra o projeto no VS Code.

Passo 3: Fazer Commits no Repositório

  1. Faça alterações no código.
  2. Vá para a aba Source Control (ícone de ramos).
  3. Clique no ícone + para adicionar os arquivos.
  4. Escreva uma mensagem de commit e clique no botão de check.
  5. Clique em Sync Changes para enviar ao GitHub.

Passo 4: Trabalho em Equipe

  1. Cada membro deve clonar o repositório.
  2. Crie uma nova branch com Git: Create Branch.
  3. Faça alterações e commits regularmente.
  4. Crie um Pull Request na aba GitHub Pull Requests no VS Code.
  5. Sincronize as alterações com Git: Pull.

Comandos Essenciais do Git

  • git init: Inicializa um novo repositório Git em um diretório.
  • git clone: Clona um repositório existente para o seu computador.
  • git status: Mostra o estado atual do repositório (arquivos modificados, pendentes, etc.).
  • git add: Adiciona arquivos ao índice (área de preparação) para serem incluídos no próximo commit.
  • git commit: Salva as alterações localmente com uma mensagem descritiva.
  • git push: Envia as alterações locais para o repositório remoto no GitHub.
  • git pull: Baixa alterações do repositório remoto e sincroniza com sua versão local.
  • git branch: Gerencia branches no repositório (criar, listar, alternar).
  • git merge: Mescla mudanças de uma branch em outra.
  • Sync Changes: Combina git pull e git push no VS Code para sincronizar mudanças facilmente.
